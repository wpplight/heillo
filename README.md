# Heillo

Heillo是一个用Rust编写的终端用户界面(TUI)应用程序，用于管理和查看订阅内容。

## 功能特性

- 📋 **主视图**：查看所有订阅项目列表
- 📝 **详情视图**：查看订阅项目的详细信息列表
- ✏️ **编辑模式**：编辑详情项的标题、描述和文本内容
- 💾 **保存模式**：选择保存或不保存编辑的内容
- ⌨️ **键盘导航**：使用键盘快捷键快速操作
- 📌 **窗口置顶**：可以将应用窗口置顶
- 🔄 **刷新列表**：刷新订阅列表

## 安装方法

### 前提条件

- Rust 1.60+ (可通过 [rustup](https://rustup.rs/) 安装)

### 安装步骤

1. 克隆仓库：
```bash
git clone https://github.com/your-username/heillo.git
cd heillo
```

2. 构建项目：
```bash
cargo build --release
```

3. 运行应用：
```bash
cargo run --release
```

## 使用说明

### 主视图

主视图显示所有订阅项目列表。在这个视图中，你可以：

- 使用上下箭头键或 `j`/`k` 导航列表
- 按 `Enter` 查看选中项目的详情
- 按 `a` 添加新订阅项
- 按 `d` 删除选中的订阅项
- 按 `r` 刷新列表
- 按 `t` 切换窗口置顶状态
- 按 `q` 或 `Esc` 退出程序

### 详情视图

详情视图显示订阅项目的详细信息列表。在这个视图中，你可以：

- 使用上下箭头键或 `j`/`k` 导航详情列表
- 按 `Enter` 查看选中详情项的完整内容
- 按 `q` 或 `Esc` 返回主视图

### 详情页面

详情页面显示单个详情项的完整内容，包括标题、描述和文本。在这个视图中，你可以：

- 使用上下箭头键或 `j`/`k` 切换选中的编辑区域
- 按 `v` 进入编辑模式
- 按 `d` 清空选中区域的内容
- 按 `b` 模拟键盘输出（2秒后）
- 按 `q` 或 `Esc` 返回详情视图

### 编辑模式

在编辑模式下，你可以：

- 输入字符来修改文本
- 按 `Backspace` 删除字符
- 按 `Enter` 换行
- 按 `Esc` 完成编辑并进入保存模式

### 保存模式

在保存模式下，你可以：

- 按 `q` 不保存退出编辑模式
- 按 `w` 保存并退出编辑模式
- 按 `Esc` 返回编辑模式

## 快捷键列表

### 通用
- `q` 或 `Esc`：退出当前视图或程序

### 主视图
- `↑`/`↓` 或 `j`/`k`：上下导航
- `Enter`：查看项目详情
- `a`：添加订阅项
- `d`：删除订阅项
- `r`：刷新列表
- `t`：切换窗口置顶

### 详情视图
- `↑`/`↓` 或 `j`/`k`：上下导航
- `Enter`：查看详情项内容

### 详情页面
- `↑`/`↓` 或 `j`/`k`：切换编辑区域
- `v`：进入编辑模式
- `d`：清空选中区域内容
- `b`：模拟键盘输出

### 编辑模式
- 字符输入：输入文本
- `Backspace`：删除字符
- `Enter`：换行
- `Esc`：完成编辑

### 保存模式
- `q`：不保存退出
- `w`：保存并退出
- `Esc`：返回编辑模式

## 技术栈

- **Rust**：主要编程语言
- **ratatui**：终端用户界面库
- **crossterm**：终端处理库

## 项目结构

```
src/
├── app.rs          # 应用程序逻辑和状态管理
├── input.rs        # 输入处理
├── main.rs         # 程序入口点
├── types.rs        # 类型定义
├── ui.rs           # UI渲染
└── utils.rs        # 工具函数
```
